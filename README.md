## 개요

자료를 유의미하게 정리하기 위해서는 많은 시간과 노력이 필요합니다.
Recordly는 `텍스트 분석을 통한 태그 키워드 추천` 및 `연관 메모 자동 추천 기능`을 제공합니다.
디바이스와 무관하게 `웹 서비스 형태`로 Recordly를 시작해보세요

<hr>
## 팀 정보

| 분류    | 내용                                                                |
| ------- | ------------------------------------------------------------------- |
| 기간    | 2022년 6월 ~ 2023년 2월                                             |
| 소속    | 소프트웨어 마에스트로 13th                                          |
| 팀 구성 | Front-End 1명, Back-End 1명                                         |
| 소개    | 작성한 메모에 대해 태그 추천 및 메모 추천을 제공하는 웹 필기 서비스 |

<hr>
## 주요 기능

- `포스트잇`, `도형 그리기`, `펜 필기`, `텍스트`, `사용자 정의 스타일` 등 다양한 메모 에디팅 기능
- 메모 `즐겨찾기` 및 `폴더 관리`를 통한 편리한 메모 분류 기능
- 작성한 메모에 대해 `태그 추천` 기능
- 작성한 메모에 대해 `관련 메모 추천` 기능
